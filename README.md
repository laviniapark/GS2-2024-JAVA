# GS2-2024 JAVA

### Descrição  
Este é um projeto Java desenvolvido para gerenciar clientes, eletrodomésticos e cálculos relacionados ao consumo de energia elétrica. Ele utiliza serviços RESTful para operações CRUD e está integrado com uma API externa (ViaCep) para consulta de endereços.

---
### Tecnologias Utilizadas  
- Java 17 (ou a versão que você estiver usando)
- Apache Tomcat (servidor de aplicação)
- Jersey (para criar APIs RESTful)
- Postman (para testes das APIs)
- Eclipse IDE (para desenvolvimento)
